# Spring-Boot-Redis
Spring Boot 整合 Redis。

### Spring Boot 依赖
    
- Web
    - Web                   使用Tomcat和Spring MVC进行全栈Web开发
    
- Template
	- Thymeleaf             Thymeleaf模板引擎（页面）
    
- SQL
    - MySQL                 MySQL JDBC驱动程序
    
- NoSQL
    - Redis                 Redis键值数据存储，包括spring-data-redis
